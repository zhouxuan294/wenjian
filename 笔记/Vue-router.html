router-link

  • tag： router-link 默认在页面中渲染为a标签，通过属性“tag”可以更改 router-link 在页面中渲染后展示的标签名
1.//例如：

2.<router-link to='/home' tag='button'>首页</router-link>;//按钮

3.<router-link to='/list' tag='li'>列表</router-link>;//li标签
  • active-class：控制当前选中状态的元素的类名，该属性的属性值就是对应元素在选中状态下的类名，不写这个属性的话，默认值是“ router-link-active ”
1.<router-link active-class='current'></router-link>
  • linkActiveClass：修改全局中的选中状态下的“router-link”的类名，写在VueRouter实例中
1.let router=new VueRouter({

2.    linkActiveClass:'current_1',

3.    routes,

4.})
  • exact-active-class：精确匹配路由（会把一些参数识别为不同的路径），不写这个属性的话，默认值是“ router-link-exact-active ”
1.<router-link exact-active-class='exactClass'></router-link>
  • linkExactActiveClass：修改全局中的精确匹配路由类名，写在VueRouter实例中
1.let router=new VueRouter({

2.    linkExactActiveClass:'activeClass',

3.    routes,

4.})
  • query：后边的对象中的每一项都会被拼接到路由后边
1.<router-link :to='{path:'/home',query:{a:1,b:2}}'>文字内容</router-link>
  • $route.query：获取通过query传递的信息
1.let home={

2.    template:'<h2>首页{{$route.query}}</h2>',

3.    created(){

4.        console.log(this.$route.query);

5.    }

6.}
  • params：后边的对象中的参数会被当作路径拼接到原路径上
1.<router-link :to='{name:'XXX',params:{c:1,D:2}}'>文字内容</router-link>

path:'/list' 必须要改为 name:'XXX'，name是固定写法，XXX是自定义，而对于“query”，两种写法都可以。要在对应的路由列表中声明：

1.let routers=[

2.    {

3.        path:'/list/:c/:d',

4.        name:'XXX',

5.        component:list,

6.    }

7.]
  • $route.params：获取通过params传递的信息
1.let list={

2.    template:'<h2>列表{{$route.params}}</h2>',

3.    created(){

4.        console.log(this.$route.params);

5.    }

6.}

2.    <router-view></router-view>
