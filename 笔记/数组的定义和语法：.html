数组的定义和语法：

Edit

数组的定义和语法：

  • 数组对象是使用单独的变量名来存储一系列的值。数组的每一项都有一个属性名（数组本身就是一个对象，他只是对象的一个细分）跟他对应，这里的属性名（依次递增的数字）叫索引；数组的每一项可以是任意数据类型。
  • 创建一个数组：

1.常规方式：

var myCars=new Array();

myCars[0]='Saab';

myCars[1]='Volvo';

myCars[2]='BMW';


2.简洁方式：

var myCars=new Array('Saab',"Volvo",'BMW');

3.字面 :

var myCars=['Saab','Volvo','BMW'];

  • 访问查看数组（通过指定数组名以及索引号码）：
var name=myCars[0];

myCars[0]='Opel';(修改了第一个元素)


1.for/in循环查看数组：

for(var k in ary){

    console.log(ary[k]);

    }


2.for循环查看数组：

for(var i=0;i<ary.length;i++){

    console.log(ary[k]);

    }


3.while循环查看数组：

var n=0;

while(n<ary.length){

    console.log(ary[n]);

    n++;

    }

  • 数组中元素数量的查看：
var ary=[1,2,3,a,b,c];

var a=ary.length;

console.log(a);//6
