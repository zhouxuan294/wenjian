图片懒加载：

  • window.onresize：监听浏览器可视窗口的变化
  • window.onscroll：监听页面滚动事件
  • 图片懒加载：只有当页面滚动到图片位置显示时才会去加载图片，一开始，图片的位置可以放一张小的默认图。
    • 优势：提升页面的加载速度；应用在移动端可以省流量。
    • 实现原理：监听页面滚动，当页面划到图片位置的时候，再让图片去加载。

22.<script>

23.   function imgLoad(ele){

24.      var tarT=ele.offsetTop;

25.      var cliH=document.documentElement.clientHeight||document.body.clientHeight;

26.      var scrH=document.documentElement.scrollTop;

27.      if(scrH+cliH>tarT){

28.         ele.src='图片链接地址';

29.      }

30.   }

31.   var img=document.getElementsByTagName('img')[0];

32.   window.onscroll=function(){

33.      imgLoad(img);

34.   }
