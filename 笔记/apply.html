apply

apply与call用法基本相同。不同的是：apply的参数中，从第二项开始，必须是数组或类数组，不能是数字；call的参数中，从第二项开始，可以是其他数据类型。

1.function sum(){

2.   console.log(this);

3.   return arguments;

4.}

5.function sum2(){

6.   return 2;

7.}

8.sum.apply(sum2,[1,2,3]);//f sum2(){...} [1,2.3]

9.sum.apply(sum2,[2],[1]);//f sum2(){...} [2]。不接收第三个参数

1.function sum(n,m){

2.   return n+m;

3.}

4.function sum2(){

5.   return 2;

6.}

7.sum.apply(sum2,[1,2]);//3

8.sum.apply(sum2,[1],[2]);//NaN
