数组部分常用方法：

  • 会改变原数组的方法：pop , push , shift , unshift , reverse , sort , splice
  • 不改变原数组的方法：slice , concat , toString , join , indexOf
  • 迭代的方法：【for(let i=0;i>ary.length;i++){…}：编程式；ary.forEach(()=>{…})：声明式】
    • forEach
    • map
    • filter【过滤，不改变原数组。循环当前项时，若return的值是true，则把当前项复制一份返回给新数组；若return的值是false，则不返回任何数据】
1.let ary=[1,2,3,4,5];

2.let arr=ary.filter((item,index)=>{

3.    return true;//arr与ary内容相同

4.    return false;//arr=[]

5.    return item>3;//arr=[4,5]

6.

9.})
    • some【返回的是布尔数据类型（true或false），循环项目中，只要有一项满足条件，返回的就是true；全都不满足条件时，返回的是false。只要出现一项是true，就不再继续循环】
1.let ary=[1,2,3,4,5];

2.let boolean=ary.some((item,index)=>{

3.    let t=Matn.random();

4.    return t>0.5;

5.});
    • every【返回的是布尔类型数据（true或false），循环项目中，都满足条件时返回的是true，只要有一项不满足条件，返回的就是false，而且只要出现一个false，就不再继续循环】
1.let ary=[1,2,3,4,5];

2.let boolean=ary.every((item,index)=>{

3.    let t=Math.random();

4.    return t>0.5;

5.});
    • find【返回值是符合条件的第一项，没有符合条件的时候，返回值是“undefined”】
1.let ary=[1,2,3,4,5];

2.let number=ary.find((item,index)=>{

3.    return item>3;

4.});

5.//number=4
    • includes( [item] , [index] )【返回的是布尔类型数据（true或false），[item] 是要查找的项，从索引为[index]的那一项开始查找，若该项在数组中，返回值就是“true”，不在，就返回“false”】
1.let ary=[1,2,3,4,5];

2.let boolean_1=ary.includes(3,0);//true

3.let boolean_2=ary.includes(3,3);//false
    • reduce( ( [prev] , [next] , [index] , [ary] )=>{} , [x] )【返回值是最后一次迭代的返回值。第一次循环时：如果传了第二个参数[x]，则[prev] = [x]，如果没有传[x]，则默认[prev]为原数组的第一项；如果传了第二个参数[x]，则[next]为原数组的第一项，如果没有传[x]，则[next]为原数组的第二项；[index]是[next]的索引；[arr]是所循环的原数组。从第二次循环开始，每次循环中的[prev]都是上次循环return的值，若没有return值，则[prev]就是“undefined”】
1.let ary=[2,4,6,8];

2.let a=ary.reduce((prev,next,index,arr)={

3.    console.log(prev,next,index,arr);

4.});
