函数的递归求和：

1、求[0,n]之间的所有的整数的和；

function sum(n){

    if(n<=0){

        return 0;

        }

    return n+sum(n-1)

    }


sum(6);//0+1+2+3+4+5+6=21


2、求[0,n]之间的奇数和：

function sum(n){

    if(n<=0){

        return 0;

        }

    if(n%2==1){

        return n+sum(n-2);

        }

    }


sum(9);//1+3+5+7+9=25


上述代码中，n只能是奇数，n为偶数时函数输出为undefined。

function sum(n){

    if(n<=0){

        return 0;

        }

    if(n%2==1){

        return n+sum(n-2);

        }else{

        return sum(n-1)}

    }


上述代码中，n既可以是偶数，也可以是奇数。
