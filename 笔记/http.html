http

  • 建立一个网站需要的内容：
    • 首先需要一个服务器，建立一个服务，会对应的给一个IP地址（32位二进制地址）一般看到的IP地址都是点位十进制（例如：198.168.1.1）
    • 使用ftp上传本地项目
    • 去万网上买一个域名，把域名跟你的IP地址联系起来。外界就可以直接通过域名访问你的网址
      • .com：国际商用公司
      • .cn：中国境内使用
      • .net：网络公司
      • .gov：一般是一个政府官网
      • .org：一般是一个官方组织
      • .edu：一般是教育机构
        https://news.163.com
        .com：顶级域名（一级域名）
        163：二级域名
        news：三级域名
  • 在浏览器地址地址栏输入地址后，敲回车，经历的过程：
    如果想访问一个服务器，必须要知道的就是这个服务器的IP地址；
    DNS（domain name system）域名系统，把访问的域名对应的IP地址返回
    • 输入网址后，浏览器（客户端）会先去自己的缓存中查找对应的IP，若没有对应的IP，就去本地区的DNS服务器中查找对应的IP，若还没有，就再去上级DNS中查找，直到找到这个域名对应的IP地址；返回的时候，浏览器会对这个域名进行缓存，本地区的DNS服务器也会做一个缓存，下次本机再去访问的时候，直接从本地获取IP；
    • 通过对应的IP地址访问对应的服务器；
    • 服务器通过对应的路径，把客户端请求的内容返回给客户端（.html/.css 这些都是把源码返回给客户端）；
    • 客户端接收到源代码后，会先进性DOM树的计算，然后再去根据加载的CSS文件渲染DOM树，最后再根据加载过来的JS对DOM树进行修改
      （从请求开始到请求结束的整个过程称为一次“http事务”。一个网页的加载要经历多次http事务）
  • http报文：
    一次请求有时候需要我们给后台（服务器）传递一些参数，后台返回的时候，也会返回给我们很多数据，我们把这些请求的数据统称为“http报文”。
    • 通用头（General）：
      Request URL:https://static.souyidai.com/wap/2.0/styles/main.css【请求路径】
      Request Method:GET 【请求方式】
      Status Code:200 ( from disk cache ) 【响应状态（从本地获取的）】
      Referrer Policy:no-referrer-when-downgrade
    • 响应头（Response Headers）：
      access-control-allow-headers:Origin,Rangs,Content-Type
      access-control-allow-methods:GET,POST,OPTIONS 【支持的请求方式】
      access-control-allow-origin:* 【支持跨域】
      cache-control:max-age=604800 【缓存时间（秒）】
      content-encoding:gzip 【支持的压缩格式】
      content-type:application/javascript;charset=utf-8 【返回的文件格式】
      date:Tue, 04 Sep 2018 01:52:40 GMT 【服务器响应时间】
      expries:Tue, 11 Sep 2018 01:52:40 GMT 【缓存结束时间】
      last-modified: Fri, 29 Jun 2018 04:19:01 GMT 【上次的修改时间】
      server:SWS
      status:200
    • 请求头（Request Headers）：
      accept: application/json, text/plain, 【接收的数据格式】
      accept-language:zh-CN,zh;q=0.9 【接收的数据语言类型】
