break和continue:

  • break是跳出整个循环：
for(i=0;i<10;i++){

     if(i==5){

         break;

         }

     console.log(i);

     }

  • continue是跳出本次循环：
for(i=0;i<10;i++){

    if(i==5){

        continue;

        }

    console.log(i);

    }
