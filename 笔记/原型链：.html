原型链：

Edit

原型链：

  • 作用域链：针对变量的一种查找机制，查看是不是自身私有的，不是的话再去上级作用域查找…
  • 原型链：针对属性或者方法的一种查找机制，先在自己的私有属性上产找，没有的话通过__proto__向上级共有属性中查找。
    • 私有属性：当前实例自身的，属于它的私有属性；
    • 共有属性：在实例所属的原型链上（排除实例自身的属性）的属性。
    • JS中所有的原型链的重点都是Object.prototype，也就是说，Object.prototype上的方法，所有数据类型都可以使用。
  • instance：查看该变量的类型在不在对应的原型链上。
1.var ary=[];

2.console.log(ary instanceof Array);//true

3.console.log(ary instanceof Object);//true
  • hasOwnProperty ：查看属性是否是私有的。
1.function Person(){

2.   this.name='abc';

3.   this.age=12;

4.}

5.Person.prototype.getName=function(){

6.   console.log(this.name);

7.};

8.var per1=new Person();

9.per1.getName();

10.console.log(per1.hasOwnProperty('name'));//true

11.console.log(per1.hasOwnProperty('getName'));//false
  • in：判断属性是否属于某个对象，不区分私有属性和共有属性，可以调用到的方法或者属性用in判断结果都是true。
1.console.log('getName' in per1);//true

2.console.log('toString' in per1);//true

利用函数实现push的功能，并将该函数添加到原型链中

  • 添加一项：
1.function myPush(n){

2.   this[this.length]=n;

3.   return this.length;

4.}

5.[].__proto__.myPush=myPush;

6.//或者：

7.Arry.prototype.myPush=myPush;
  • 添加多项：
1.function myPush(){

2.   for(let i=0;i<arguments.length;i++){

3.      this[this.length]=arguments[i];

4.   }

5.   return this.length;

6.}

7.[].__proto__.myPush=myPush;

8.//或者：

9.Array.prototype.myPush=myPush;

判断某个属性是否是共有属性

1.Object.prototype.hasPublicProperty=function(str){

2.   if(str in this&&!this.hasOwnProperty(str)){

3.      return true;

4.   }

5.   return false;

6.}

7.//--上下两段代码等价--

8.Object.prototype.hasPublicProperty=function(str){

9.   return str in this&&!this.hasOwnProperty(str);

10.}
